CopyNod

Did that actually copy?

A tiny menu bar utility that shows a check mark only when ⌘C actually copied something — so you never paste just to find out.

HUD demo — looping

The problem

You press ⌘C. The screen says nothing — it never does. So you paste somewhere just to check, or copy it three more times to be safe.

How it works

A keypress is only a trigger. The HUD appears only after the clipboard change counter actually changes — nothing copied, nothing shown.
